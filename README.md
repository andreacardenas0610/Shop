💅 NailBeauty Store

NailBeauty Store es una tienda web desarrollada con React y Material UI que permite a los usuarios explorar productos de belleza para uñas, agregarlos al carrito de compras, marcarlos como favoritos y simular un proceso de compra. La aplicación cuenta con un diseño moderno e intuitivo que se adapta a diferentes dispositivos como computadores, tablets y teléfonos móviles gracias a su diseño responsive.

La plataforma incluye varias funcionalidades principales como la visualización de productos con imagen, nombre, descripción y precio, un sistema de favoritos para guardar artículos de interés, y un carrito de compras donde los usuarios pueden agregar o eliminar productos, vaciar el carrito y ver el total de la compra. Además, se incluye una sección de compras donde se puede visualizar el historial después de realizar el pago.

El proyecto fue desarrollado utilizando React, Material UI, React Router DOM y Context API para manejar el estado global del carrito y los favoritos. La estructura del proyecto está organizada en componentes, páginas y contextos para facilitar el mantenimiento del código.

Este proyecto funciona como una simulación de una tienda en línea y puede ampliarse en el futuro con funcionalidades adicionales como búsqueda avanzada, autenticación de usuarios o integración con sistemas reales de pago.
