💅 NailBeauty Store
Descripción

NailBeauty Store es una tienda web desarrollada con React y Material UI que permite a los usuarios explorar productos de belleza para uñas, agregarlos al carrito de compras, marcarlos como favoritos y simular un proceso de compra. La aplicación cuenta con un diseño moderno, intuitivo y responsive, lo que permite que funcione correctamente en diferentes dispositivos como computadores, tablets y teléfonos móviles.

Características principales

La aplicación incluye varias funcionalidades que mejoran la experiencia del usuario. Entre ellas se encuentran la visualización de productos con imagen, nombre, descripción y precio, un sistema de favoritos que permite guardar productos de interés, y un carrito de compras donde los usuarios pueden agregar o eliminar productos, vaciar el carrito y visualizar el total de la compra. Además, la aplicación permite simular el proceso de pago mostrando un mensaje de confirmación de compra.

Interfaz gráfica

La interfaz gráfica fue desarrollada utilizando Material UI, lo que permite crear un diseño limpio, moderno y fácil de usar. La navegación se realiza a través de un Navbar que incluye acceso a las diferentes secciones del sitio como inicio, productos, ofertas, cuenta, favoritos y carrito. En dispositivos móviles el menú se adapta automáticamente utilizando un menú hamburguesa, lo que mejora la experiencia de navegación en pantallas pequeñas.

Arquitectura del proyecto

El proyecto está organizado utilizando una estructura basada en componentes para facilitar el mantenimiento del código. Se utilizan componentes reutilizables para la navegación y la visualización de productos, páginas para cada sección principal de la aplicación y Context API para manejar el estado global del carrito de compras y los favoritos. La estructura del proyecto separa claramente los componentes, páginas y contextos, permitiendo una mejor organización y escalabilidad del sistema.
